Project with APIs

The project has been created on Flask Micro Web Framework. SQL Alchemy has been used as the ORM for the application. The app python file, as you might have realised by now, is the start point of this application. As mentioned in the description API Collect and API Similar has been captured.

In addition to these files, I have introduced a new file and route to add new city and the associated latitude and longitude details. The Add City requires , you to enter the details, which will be saved in the database. This has facilatated the expansion of the application.
